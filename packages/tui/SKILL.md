# @tokmeter/tui

Interactive terminal UI for token usage tracking. Full-screen dashboard with charts, sparklines, and heatmaps.

## Capabilities

- Overview with bar charts and sparklines
- Model comparison table with inline charts
- Daily usage trend with contribution heatmap
- Statistics view with streaks and averages
- Keyboard navigation (1-4 views, Tab, arrows, q to quit)

## Usage

```bash
npx @tokmeter/tui
```
